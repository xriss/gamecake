This example demonstrates a complete NaCl PPAPI application. In source file hello_world.c contains the three required PPAPI functions PPP_InitializeModule, PPP_GetInterface, PPP_ShutdownModule which initialize the application, provide a string name to callback interface mapping, and cleanup on shutdown respectively.

The example will query for the Console, PostMessage and Var interfaces to send a message to JavaScript which will be added to the output box. In addition, it will send another message to the JavaScript Console to simulate logging for development.
